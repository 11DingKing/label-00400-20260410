# 赛事管理系统

## How to Run

```bash
# 使用 Docker Compose 启动所有服务
docker-compose up --build -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

## 题目内容
第一部分：前端开发要求一、核心目标构建用户体验优良、视觉美观、响应式的前端界面，实现文章管理、赛事报名、成绩查询等核心功能的交互，与后端接口高效联调，保障不同角色操作的权限适配。二、技术栈要求核心框架 ：React（推荐 18 + 版本）；UI 组件库 ：Ant Design（推荐 5 + 版本）；路由 ：React Router（6 + 版本）；HTTP 请求 ：Axios（封装请求拦截、响应拦截）；富文本编辑器 ：React Quill（或其他成熟富文本组件）；构建工具 ：Vite（或 Create React App）；适配 ：支持响应式设计，兼容 PC、平板、手机等设备。三、功能模块要求1. 通用模块1.1 导航与布局顶部导航栏 ：包含首页、文章列表、成绩查询等公共入口，管理员角色额外显示「发布文章」「报名列表」「成绩管理」等入口；权限适配 ：根据后端返回的用户权限，动态显示 / 隐藏菜单和操作按钮；页脚 ：展示版权信息、联系方式，样式简洁且适配不同屏幕。1.2 全局交互请求拦截 ：所有接口请求添加 Loading 状态（如页面级 / 按钮级 Loading）；响应处理 ：统一捕获接口错误，弹出提示框（如参数错误、权限不足、服务器错误）；Token 管理 ：登录后存储 JWT Token 到 localStorage，请求头自动携带 Token，Token 过期时跳转至登录页。2. 核心业务模块2.1 首页赛事报名区 ：展示赛事卡片（包含赛事名称、简介、背景图），点击「立即报名」跳转至对应赛事的报名页（携带赛事 ID）；文章展示区 ：展示精选文章列表（标题、简介、作者），支持分页和点击查看详情；成绩查询入口 ：显眼的按钮，跳转至成绩查询页。2.2 文章管理模块文章发布页 ：包含标题输入框、作者输入框、富文本编辑器（支持图文混排）；表单校验（标题必填），提交后提示结果并重置表单；文章列表页 ：支持分页、标题模糊搜索，展示文章标题、作者、发布时间；文章发布者仅能看到自己的文章编辑 / 删除按钮，超级管理员可操作所有文章；文章详情页 ：完整展示文章内容，样式排版美观。2.3 报名系统模块动态背景 ：根据赛事 ID 自动切换背景图（默认背景兜底），背景图做降亮度处理；报名表单 ：包含姓名、手机号（格式校验）、身份证号（格式校验）、赛事类型（下拉选择）；提交前校验唯一性（调用后端校验接口），防止重复报名；报名列表页（仅报名录入者可见） ：支持分页、按身份证号 / 姓名搜索，可导出报名数据（Excel 格式）。2.4 成绩查询与管理模块成绩查询页（公共） ：仅需输入身份证号，点击查询展示对应赛事成绩（无数据时提示）；成绩管理页（仅成绩发布者可见） ：支持录入 / 更新成绩（身份证号、赛事 ID、成绩、排名）；展示成绩列表，支持分页、搜索，可导出成绩数据。3. 权限适配要求登录页 ：包含账号、密码输入框，登录成功后获取用户权限并跳转至首页；权限渲染 ：根据后端返回的 /api/user/permission 接口数据，动态渲染菜单和操作按钮；无权限拦截 ：用户手动访问无权限页面时，跳转至 403 页面。四、非功能要求性能 ：首屏加载时间≤3 秒，二次加载≤1 秒，图片懒加载；兼容性 ：支持 Chrome、Firefox、Edge、Safari 最新版，IE11 及以上；可维护性 ：代码模块化，组件拆分合理，添加必要注释；用户体验 ：操作反馈及时，表单校验友好，错误提示清晰。五、交付物要求完整的前端项目源码（含 node_modules 除外的所有文件）；项目运行文档（环境配置、启动命令、打包步骤）；组件使用说明（核心组件的功能和使用方法）。
### 核心目标

构建用户体验优良、视觉美观、响应式的前端界面，实现文章管理、赛事报名、成绩查询等核心功能的交互。

### 技术栈

- 核心框架：React 18+
- UI 组件库：Ant Design 5+
- 路由：React Router 6+
- HTTP 请求：Axios
- 富文本编辑器：Tiptap
- 构建工具：Vite

### 功能模块

1. **通用模块**
   - 顶部导航栏：根据用户权限动态显示菜单
   - 全局交互：请求拦截、响应处理、Token 管理

2. **首页**
   - 赛事报名区：展示赛事卡片，点击跳转报名页
   - 文章展示区：展示精选文章列表
   - 成绩查询入口

3. **文章管理模块**
   - 文章发布页：标题、作者、富文本编辑器
   - 文章列表页：分页、搜索、编辑/删除
   - 文章详情页：完整展示文章内容

4. **报名系统模块**
   - 报名表单：姓名、手机号、身份证号、赛事类型
   - 报名列表页：分页、搜索、导出

5. **成绩查询与管理模块**
   - 成绩查询页：输入身份证号查询成绩
   - 成绩管理页：录入/更新成绩、导出

## 项目介绍

本项目是一个现代化的赛事管理系统前端，采用 React 18 + Ant Design 5 + Vite 技术栈开发，具有以下特点：

- 🎨 现代化 UI 设计，参考 Nike Run Club、Strava 等顶级运动网站风格
- 📱 响应式布局，支持 PC、平板、手机等设备
- 🔐 完善的权限管理，支持多角色访问控制
- ⚡ 基于 Vite 构建，开发体验极佳
- 🐳 Docker 容器化部署，一键启动
